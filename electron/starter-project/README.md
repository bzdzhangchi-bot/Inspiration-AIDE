# Inspiration Starter Project

欢迎来到 Inspiration。

这个内置项目是给第一次打开应用的用户准备的，目的是让你不用先找本地仓库，也能马上开始熟悉界面和工作流。

建议先打开这几个文件：

- `新手引导.md`：快速了解这个应用怎么用
- `pom.xml`：看看内置的 Spring Boot 项目结构
- `src/main/java/com/inspiration/starter/StarterWorkspaceApplication.java`：应用入口
- `src/main/java/com/inspiration/starter/StarterWorkspaceController.java`：最小 REST Controller
- `src/main/java/com/inspiration/starter/StarterWorkspaceService.java`：简单业务类
- `src/main/resources/application.properties`：看看配置文件会放在哪里
- `examples/preview-demo.html`：体验 HTML 预览
- `examples/mermaid-demo.md`：体验 Markdown + Mermaid 预览

你也可以直接在这个项目里新建文件、改内容、试终端命令，熟悉之后再切换到自己的真实项目。
